# AI Sudoku Solver - Search Algorithms

This project implements various AI Search Algorithms to solve Sudoku puzzles as part of the Level 3 AI Course.

## Algorithms Implemented
- **Uninformed Search:** BFS, DFS (Backtracking).
- **Informed Search:** A* (using MRV Heuristic).
- **Optimization:** Hill Climbing, Genetic Algorithm.

## Project Structure
- `src/`: Contains source code.
- `docs/`: Project report and documentation.
- `results/`: Performance comparisons and screenshots.

## How to Run
1. Install Python 3.x.
2. Run the main file: `python src/main.py`
